    Face Recognition Attendance System

    Face recognition is a practical application of pattern recognition
    theory, whose task is to automatically localize a face in a photo and,
    if necessary, identify a person by face.

    How it works

    When you unlock your phone without a password using the camera, you use this technology. The method is applied biometric comparison, which with the help of machine learning models
    allows you to compare a face with a database and find coincidences. Face recognition can occur as on a video stream,
    when a person is captured by a camera (for example, a smart intercom), so and by a static image (for example, when social networks offer you to mark yourself in other people's photos)
